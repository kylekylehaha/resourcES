三大功能讓你輕鬆上手

剩下多少，一覽即知。

清楚記錄既有的資源數量，方便使用者快速查詢，並即時更新等候人數。

乖乖排隊，不要插隊。

完善的預約系統，以及相對應的懲罰機制，可以維持使用者的品質，讓你可以如期借用到器材。

開放資源，人人開放。

結合資料庫運用，提出概念模型

Schema

Schema 是一種用來描述一種結構，是一種對象的集合。透過 schema，可以讓開發人員更清楚的了解，資料庫內資料的儲存方式，這樣未來維護時可以快速了解資料型態。
